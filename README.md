### 3. AppLink 配置演示
使用这条短链可以下载或者直接打开 app 的指定页面 https://gio.ren/PQ2r6aaidoMo ， 没有安装应用将跳转安装，有应用则会直接打开 app 指定页面，携带自定义参数。
可以将短链放在 短信、邮件等应用中，点击短链则会直接打开指定页面。
